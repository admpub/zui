    Not support file:// protocol

    You are using file:// protocol to access ZUI documents. ZUI documents need to be loaded asynchronously, so you need to use http:// or https:// to access it.

    If you downloaded the whole ZUI source code package and the system is already installed Node.js, follow the steps below to get access to your documents right away.

    1. Enter ZUI source code directory;
    2. Execute npm install;
    3. Execute npm start；
    4. Visit http://127.0.0.1:8088。

    You can also access online documentation:
